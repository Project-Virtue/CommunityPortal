# Community Portal
This repository is for the [ForcesOfLight](https://github.com/Project-Virtue/ForcesOfLight) community portal.

# Development

This project is built on the Django python framework.

## Getting Started

1. Bootstrap the project:
    - 
    ```bash
    make bootstrap
    ```
1. Build the project:
    -
    ```bash
    make build
    ```
1. Run tests:
    - 
    ```bash
    make test
    ```
